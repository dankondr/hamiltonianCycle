# Hamiltonian Cycle #
**This program helps user to identify is there a hamiltonian cycle in loaded graph.**
<h2>How to use</h2>
Adjacency matrix should be in .txt file and be formatted as stated below.
<br />**Adjacency matrix format:**
```
Graph1 //graph name
16 //number of vertex
0111100000000000
1000010101000000
1000001010100000
1000001100010000
1000010010001000
0100100000000101
0011000000000110
0101000010100000
0010100101000000
0100000010001001
0010000100010010
0001000000100101
0000100001000110
0000011000011000
0000001000101001
0000010001010010
```
